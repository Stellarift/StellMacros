# StellMacros
Лабораторная работа: Макросы препроцессора C++. 
Цель Изучение директивы препроцессора #define и создание практических макросов для базовых операций.

Файлы проекта
macros.h - Заголовочный файл, содержит объявления всех макросов: MIN, MAX, SQUARE, IS_EVEN, IS_ODD и прототип функции power().

functions.cpp - Реализационный файл, содержит функцию power() для возведения в степень.

StellMacros.cpp - Основной файл, тестирует все макросы на практике и выводит результаты.

Описание программы
Программа создает набор макросов для выполнения математических операций и проверки свойств чисел. Макросы работают через текстовую подстановку на этапе препроцессора до компиляции кода.

Реализованные возможности
Поиск минимального и максимального из двух чисел

Возведение числа в квадрат

Возведение числа в степень

Проверка числа на четность и нечетность

Технические особенности
Использована директива #define для создания макросов, разделение кода на заголовочные и реализационные файлы, защита от повторного включения заголовочных файлов.
